Study Case

Membuat Wire frame

langkah awal yang saya lakukan sebelum saya mendevelop design ke code saya membuat wire frame terlebih dahulu karna itu juga membantu saya agar bisa bekerja dengan tim nantinya, selain itu membuat wire frame terlebih dahulu memang sangatlah berguna dan mengurangi waktu kita untuk melamun memikirkan design dan layout.

Setelah wireframe sudah selesai saya lanjutkan mendevelop tampilan web Apps yang saya buat, project ini saya selesaikan 2 hari yaitu 1 hari untuk UI design dan 1 hari untuk mendevelop tampilannya. Saya sadar masih banyak kekurangan di web apps yang saya buat ini, berikut link web apps yang saya kerjakan: https://trlmks.github.io/onlineshop/ atau Click disini!
