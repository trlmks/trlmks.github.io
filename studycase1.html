Pada tanggal 05 agustus 2021 saya pribadi mendevelop Web Apps. Tujuan saya mendevelop Web Apps selain untuk mengasah kemampuan saya, saya juga ingin memamerkan karya saya yang saya buat dengan kesungguhan.

Tema yang saya gunakan saat ini adalah Online Store karena pada era sekarang semua dituntut serba cepat dan instan tanpa harus menghabiskan waktu dan tenaga untuk berbelanja secara ofline terutama dimasa – masa sepeti ini saat pandemic Covid-19 berlangsung.

Pertama – tama sebelum saya coding Web Apps yang saya buat, saya terlebih dahulu membuat wireframe untuk membantu saya menyiapkan pembuatan web apps yang saya buat, sekalian melatih saya untuk bekerja Bersama tim seperti UI Designer agar apa yang Didesign oleh UI designer dapa saya aplikasikan Kedalam code.
